Aspecto Descrição
O que é Java? Java é uma linguagem de programação de propósito geral, orientada a objetos e baseada em classes.
História Desenvolvida pela Sun Microsystems em 1995, agora mantida pela Oracle Corporation.
Principais características Portabilidade, robustez, segurança, e suporte a multithreading.
Usos comuns Desenvolvimento de aplicativos web, aplicações corporativas, e desenvolvimento de aplicativos Android.
